Complete Python Course for Beginners

Course Overview

This course is designed for absolute beginners with no prior programming experience. By the end, you'll be able to write Python programs, understand core programming concepts, and build simple projects. The course is hands-on, with examples and exercises to reinforce learning.

  • Duration: Approximately 4-6 weeks (assuming 2-3 hours per week)
  • Prerequisites: None
  • Tools Needed:
    • Python (version 3.x) installed (download from python.org)
    • A code editor (e.g., VS Code, PyCharm, or IDLE)
    • Optional: Online platforms like Replit or Google Colab for practice

Module 1: Introduction to Python

1.1 What is Python?

Python is a high-level, versatile programming language known for its readability and simplicity. It is used for web development, data analysis, automation, AI, and more.

1.2 Installing Python

Download Python from python.org. Verify installation by running python --version in your terminal/command prompt.

1.3 Writing Your First Program

Open a code editor and create a file named hello.py. Write and run the following code:

print("Hello, World!")

Explanation: The print() function outputs text to the console.

1.4 Using an IDE or Editor

Install Visual Studio Code or PyCharm for a better coding experience. Practice running hello.py from the editor or terminal (python hello.py).

Exercise:
  1. Write a program that prints your name.
  2. Modify the program to print "Hello, [Your Name]!".

Module 2: Python Basics

2.1 Variables and Data Types

Variables store data (e.g., numbers, text). Common data types: integers (int), floating-point numbers (float), strings (str), booleans (bool).

name = "Alice"  # String
age = 25        # Integer
height = 5.5    # Float
is_student = True  # Boolean

print(name, age, height, is_student)

2.2 Basic Operations

Arithmetic: +, -, *, /, // (integer division), % (modulus), ** (exponentiation).

a = 10
b = 3
print(a + b)  # Addition: 13
print(a * b)  # Multiplication: 30
print(a // b) # Integer division: 3
print(a % b)  # Modulus: 1
print(a ** 2) # Exponentiation: 100

2.3 Strings

Strings are sequences of characters enclosed in quotes (' or "). String operations: concatenation (+), repetition (*), slicing.

greeting = "Hello"
name = "Bob"
message = greeting + ", " + name + "!"  # Concatenation
print(message)  # Output: Hello, Bob!
print(greeting * 2)  # Output: HelloHello
print(name[0])  # Output: B (first character)

2.4 Input from Users

Use input() to get user input.

name = input("Enter your name: ")
print("Hello, " + name + "!")

  3. Write a program that takes a string input and prints it in reverse.

Module 3: Control Flow

3.1 Conditionals (if, elif, else)

Use if statements to make decisions.

age = int(input("Enter your age: "))
if age >= 18:
    print("You are an adult.")
elif age >= 13:
    print("You are a teenager.")
else:
    print("You are a child.")

3.2 Comparison and Logical Operators

Comparison: ==, !=, <, >, <=, >=
Logical: and, or, not

score = 85
if score >= 90 and score <= 100:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
else:
    print("Grade: C or below")

3.3 Loops

For Loop: Iterate over a sequence (e.g., range, list).

for i in range(5):  # Prints 0 to 4
    print(i)

While Loop: Repeats as long as a condition is true.

count = 1
while count <= 5:
    print(count)
    count += 1

3.4 Break and Continue

break: Exits a loop.
continue: Skips the current iteration.

for i in range(10):
    if i == 5:
        break  # Stops at 5
    print(i)

for i in range(10):
    if i % 2 == 0:
        continue  # Skips even numbers
    print(i)
Exercises:

  2. Create a program that prints all numbers from 1 to 10 except 7.
  3. Write a program that asks for a password and allows 3 attempts to enter the correct one ("secret").

Module 4: Data Structures

4.1 Lists

Lists are ordered, mutable collections.

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # Output: apple
fruits.append("orange")  # Add item
fruits.remove("banana")  # Remove item
print(fruits)  # Output: ['apple', 'cherry', 'orange']

4.2 Tuples

Tuples are ordered, immutable collections.

coordinates = (3, 4)
print(coordinates[0])  # Output: 3

4.3 Dictionaries

Dictionaries store key-value pairs.

student = {"name": "Alice", "age": 20, "major": "CS"}
print(student["name"])  # Output: Alice
student["grade"] = "A"  # Add key-value pair
print(student)  # Output: {'name': 'Alice', 'age': 20, 'major': 'CS', 'grade': 'A'}

4.4 Sets

Sets store unique, unordered items.

numbers = {1, 2, 2, 3}
print(numbers)  # Output: {1, 2, 3}
numbers.add(4)
print(numbers)  # Output: {1, 2, 3, 4}

Module 5: Functions

5.1 Defining Functions

Use def to create a function.

def greet(name):
    return "Hello, " + name + "!"

print(greet("Alice"))  # Output: Hello, Alice!

5.2 Parameters and Return Values

Functions can take multiple parameters and return values.

def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # Output: 8

5.3 Default Parameters and Keyword Arguments

def introduce(name, age=18):
    return f"{name} is {age} years old."

print(introduce("Bob"))  # Output: Bob is 18 years old.
print(introduce(name="Alice", age=20))  # Output: Alice is 20 years old.

5.4 Scope

Variables defined inside a function are local.

x = 10  # Global
def my_function():
    x = 5  # Local
    print(x)  # Output: 5

my_function()
print(x)  # Output: 10

Module 6: Working with Files

6.1 Reading Files

Use open() to read a file.

with open("example.txt", "r") as file:
    content = file.read()
    print(content)

6.2 Writing Files

Write or append to a file.

with open("example.txt", "w") as file:
    file.write("Hello, Python!\n")

with open("example.txt", "a") as file:
    file.write("Appending this line.\n")

Module 7: Error Handling

Handle errors gracefully using try-except.

try:
    number = int(input("Enter a number: "))
    print(100 / number)
except ZeroDivisionError:
    print("Cannot divide by zero!")
except ValueError:
    print("Please enter a valid number!")

Module 8: Modules and Libraries

8.1 Importing Modules

Use built-in modules like math and random.

import math
print(math.sqrt(16))  # Output: 4.0

import random

Module 9: Mini Projects

Project 1: Number Guessing Game

Generate a random number and let the user guess it.

import random

number = random.randint(1, 100)
attempts = 0

while True:
    guess = int(input("Guess the number (1-100): "))
    attempts += 1
    if guess == number:
        print(f"Correct! It took you {attempts} attempts.")
        break
    elif guess < number:
        print("Too low!")
    else:
        print("Too high!")

Project 2: To-Do List

Create a program to manage a to-do list.

tasks = []

def add_task(task):
    tasks.append(task)
    print(f"Added: {task}")

def view_tasks():
    if not tasks:
        print("No tasks!")
    else:
        for i, task in enumerate(tasks, 1):
            print(f"{i}. {task}")

while True:
    action = input("Enter 'add', 'view', or 'quit': ")
    if action == "add":
        task = input("Enter task: ")
        add_task(task)
    elif action == "view":
        view_tasks()
    elif action == "quit":
        break

Project 3: Simple Calculator

Build a calculator for basic operations.

def calculator():
    num1 = float(input("Enter first number: "))
    op = input("Enter operator (+, -, *, /): ")
    num2 = float(input("Enter second number: "))
    
    if op == "+":
        print(num1 + num2)
    elif op == "-":
        print(num1 - num2)
    elif op == "*":
        print(num1 * num2)
    elif op == "/":
        if num2 != 0:
            print(num1 / num2)
        else:
            print("Cannot divide by zero!")
    else:
        print("Invalid operator!")

calculator()
