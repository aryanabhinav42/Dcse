Complete C++ Course for Beginners

Course Overview

This course is designed for absolute beginners with no prior programming experience. By the end, you'll be able to write C++ programs, understand core programming concepts, and build simple projects. The course is hands-on, with examples and exercises to reinforce learning.

  • Duration: Approximately 4-6 weeks (assuming 2-3 hours per week)
  • Prerequisites: None
  • Tools Needed:
    • A C++ compiler (e.g., GCC, Clang, or MSVC). Install via MinGW (Windows) or use built-in compilers on Linux/Mac.
    • A code editor (e.g., VS Code, Code::Blocks, or Dev-C++).
    • Optional: Online platforms like OnlineGDB or Replit for practice.

Module 1: Introduction to C++

1.1 What is C++?

C++ is a powerful, general-purpose programming language known for performance and flexibility. It is used for system programming, game development, desktop applications, and more.

1.2 Setting Up Your Environment

Install a C++ compiler and editor. For Windows, use MinGW; for Linux/Mac, GCC is typically pre-installed. Verify setup by running g++ --version in the terminal.

1.3 Writing Your First Program

Create a file named hello.cpp and write the following code:

#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

Explanation: #include <iostream> includes input/output functionality, cout outputs text, and main() is the program's entry point.

Compile and Run: Use g++ hello.cpp -o hello and run with ./hello (Linux/Mac) or hello.exe (Windows).

1.4 Using an IDE

Use Code::Blocks, VS Code, or Dev-C++ for an integrated experience. Practice compiling and running hello.cpp.

Exercise:
  1. Write a program that prints your name.
  2. Modify the program to print "Hello, [Your Name]!".

Module 2: C++ Basics

2.1 Variables and Data Types

Variables store data and must be declared with a type. Common types: int, double, char, bool, string.

#include <iostream>
#include <string>
using namespace std;

int main() {
    string name = "Alice";  // String
    int age = 25;          // Integer
    double height = 5.5;   // Double
    bool is_student = true; // Boolean

    cout << name << " " << age << " " << height << " " << is_student << endl;
    return 0;
}

2.2 Basic Operations

Arithmetic: +, -, *, /, % (modulus), increment (++), decrement (--).

#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 3;
    cout << "Sum: " << a + b << endl;        // 13
    cout << "Product: " << a * b << endl;    // 30
    cout << "Division: " << a / b << endl;   // 3 (integer division)
    cout << "Modulus: " << a % b << endl;    // 1
    return 0;
}

2.3 Strings

Strings require #include <string>. Support concatenation and indexing.

#include <iostream>
#include <string>
using namespace std;

int main() {
    string greeting = "Hello";
    string name = "Bob";
    string message = greeting + ", " + name + "!";
    cout << message << endl;         // Output: Hello, Bob!
    cout << greeting[0] << endl;     // Output: H
    return 0;
}

2.4 Input from Users

Use cin for input.

#include <iostream>
#include <string>
using namespace std;

int main() {
    string name;
    cout << "Enter your name: ";
    getline(cin, name);  // Handles spaces in input
    cout << "Hello, " << name << "!" << endl;
    return 0;
}
Exercises:
  1. Create a program that asks for two numbers and prints their sum, difference, product, and quotient.
  2. Write a program that asks for a userâ€™s name and age, then prints: "Hi [Name], you are [Age] years old!"
  3. Write a program that takes a string input and prints its length.

Module 3: Control Flow

3.1 Conditionals (if, else if, else)

Use if statements for decision-making.

#include <iostream>
using namespace std;

int main() {
    int age;
    cout << "Enter your age: ";
    cin >> age;
    if (age >= 18) {
        cout << "You are an adult." << endl;
    } else if (age >= 13) {
        cout << "You are a teenager." << endl;
    } else {
        cout << "You are a child." << endl;
    }
    return 0;
}

3.2 Comparison and Logical Operators

Comparison: ==, !=, <, >, <=, >=
Logical: &&, ||, !

#include <iostream>
using namespace std;

int main() {
    int score;
    cout << "Enter your score: ";
    cin >> score;
    if (score >= 90 && score <= 100) {
        cout << "Grade: A" << endl;
    } else if (score >= 80) {
        cout << "Grade: B" << endl;
    } else {
        cout << "Grade: C or below" << endl;
    }
    return 0;
}

3.3 Loops

For Loop: Iterate over a range.

#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 5; i++) {
        cout << i << endl;  // Prints 0 to 4
    }
    return 0;
}

While Loop: Repeats while a condition is true.

#include <iostream>
using namespace std;

int main() {
    int count = 1;
    while (count <= 5) {
        cout << count << endl;
        count++;
    }
    return 0;
}

3.4 Break and Continue

break: Exits a loop.
continue: Skips the current iteration.

#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 10; i++) {
        if (i == 5) break;  // Stops at 5
        cout << i << endl;
    }
    for (int i = 0; i < 10; i++) {
        if (i % 2 == 0) continue;  // Skips even numbers
        cout << i << endl;
    }
    return 0;
}
Exercises:
  1. Write a program that checks if a number is even or odd.
  2. Create a program that prints all numbers from 1 to 10 except 7.
  3. Write a program that asks for a password and allows 3 attempts to enter the correct one ("secret").

Module 4: Data Structures

4.1 Arrays

Arrays store fixed-size sequences of elements of the same type.

#include <iostream>
using namespace std;

int main() {
    int numbers[5] = {1, 2, 3, 4, 5};
    cout << numbers[0] << endl;  // Output: 1
    numbers[1] = 10;  // Modify element
    cout << numbers[1] << endl;  // Output: 10
    return 0;
}

4.2 Vectors

Vectors (from <vector>) are dynamic arrays.

#include <iostream>
#include <vector>
using namespace std;

int main() {
    vector<string> fruits = {"apple", "banana", "cherry"};
    cout << fruits[0] << endl;  // Output: apple
    fruits.push_back("orange");  // Add item
    fruits.erase(fruits.begin() + 1);  // Remove banana
    for (string fruit : fruits) {
        cout << fruit << endl;  // Output: apple, cherry, orange
    }
    return 0;
}

4.3 Strings as Arrays

Strings can be treated as character arrays.

#include <iostream>
#include <string>
using namespace std;

int main() {
    string text = "Hello";
    cout << text[0] << endl;  // Output: H
    text += " World";  // Concatenate
    cout << text << endl;  // Output: Hello World
    return 0;
}
Exercises:
  1. Create an array of 5 integers and print the first and last elements.
  2. Write a program using a vector to store 3 names and print them.
  3. Create a program that takes a string input and prints its first and last characters.

Module 5: Functions

5.1 Defining Functions

Functions are declared with a return type and parameters.

#include <iostream>
using namespace std;

string greet(string name) {
    return "Hello, " + name + "!";
}

int main() {
    cout << greet("Alice") << endl;  // Output: Hello, Alice!
    return 0;
}

5.2 Parameters and Return Values

#include <iostream>
using namespace std;

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(5, 3);
    cout << result << endl;  // Output: 8
    return 0;
}

5.3 Default Parameters

#include <iostream>
#include <string>
using namespace std;

string introduce(string name, int age = 18) {
    return name + " is " + to_string(age) + " years old.";
}

int main() {
    cout << introduce("Bob") << endl;  // Output: Bob is 18 years old.
    cout << introduce("Alice", 20) << endl;  // Output: Alice is 20 years old.
    return 0;
}

5.4 Scope

Variables have local or global scope.

#include <iostream>
using namespace std;

int x = 10;  // Global

void myFunction() {
    int x = 5;  // Local
    cout << x << endl;  // Output: 5
}

int main() {
    myFunction();
    cout << x << endl;  // Output: 10
    return 0;
}
Exercises:
  1. Write a function that calculates the square of a number.
  2. Create a function that takes a vector of numbers and returns the maximum.
  3. Write a function that converts Celsius to Fahrenheit.

Module 6: Working with Files

6.1 Reading Files

Use <fstream> to read files.

#include <iostream>
#include <fstream>
#include <string>
using namespace std;

int main() {
    ifstream file("example.txt");
    string line;
    while (getline(file, line)) {
        cout << line << endl;
    }
    file.close();
    return 0;
}

6.2 Writing Files

Write or append to a file.

#include <iostream>
#include <fstream>
using namespace std;

int main() {
    ofstream file("example.txt");
    file << "Hello, C++!\n";
    file.close();

    ofstream appendFile("example.txt", ios::app);
    appendFile << "Appending this line.\n";
    appendFile.close();
    return 0;
}
Exercise:
  1. Write a program that creates a file and writes 5 user-input names to it.
  2. Read the file and print the names in uppercase (use toupper from <cctype>).

Module 7: Error Handling

Handle errors using exceptions with try, catch.

#include <iostream>
using namespace std;

int main() {
    try {
        int number;
        cout << "Enter a number: ";
        cin >> number;
        if (number == 0) throw "Division by zero!";
        cout << 100 / number << endl;
    } catch (const char* error) {
        cout << error << endl;
    }
    return 0;
}
Exercise:
  1. Write a program that divides two numbers and handles division by zero.
  2. Create a program that tries to read a file and handles the case where the file doesnâ€™t exist.

Module 8: Standard Template Library (STL)

8.1 Using STL Containers

The STL provides containers like vector, map, and set.

#include <iostream>
#include <map>
#include <string>
using namespace std;

int main() {
    map<string, int> student = {{"Alice", 20}, {"Bob", 22}};
    cout << student["Alice"] << endl;  // Output: 20
    student["Charlie"] = 21;  // Add key-value pair
    for (auto& pair : student) {
        cout << pair.first << ": " << pair.second << endl;
    }
    return 0;
}

8.2 Algorithms

Use STL algorithms like sort or find.

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    vector<int> numbers = {5, 2, 9, 1};
    sort(numbers.begin(), numbers.end());
    for (int num : numbers) {
        cout << num << " ";  // Output: 1 2 5 9
    }
    return 0;
}
Exercise:
  1. Write a program that uses a map to store 3 studentsâ€™ names and grades, then prints them.
  2. Create a program that sorts a vector of user-input numbers.

Module 9: Mini Projects

Project 1: Number Guessing Game

Generate a random number and let the user guess it.

#include <iostream>
#include <cstdlib>
#include <ctime>
using namespace std;

int main() {
    srand(time(0));
    int number = rand() % 100 + 1;
    int attempts = 0, guess;
    while (true) {
        cout << "Guess the number (1-100): ";
        cin >> guess;
        attempts++;
        if (guess == number) {
            cout << "Correct! It took " << attempts << " attempts." << endl;
            break;
        } else if (guess < number) {
            cout << "Too low!" << endl;
        } else {
            cout << "Too high!" << endl;
        }
    }
    return 0;
}

Project 2: To-Do List

Create a program to manage a to-do list using a vector.

#include <iostream>
#include <vector>
#include <string>
using namespace std;

vector<string> tasks;

void addTask(string task) {
    tasks.push_back(task);
    cout << "Added: " << task << endl;
}

void viewTasks() {
    if (tasks.empty()) {
        cout << "No tasks!" << endl;
    } else {
        for (int i = 0; i < tasks.size(); i++) {
            cout << i + 1 << ". " << tasks[i] << endl;
        }
    }
}

int main() {
    string action, task;
    while (true) {
        cout << "Enter 'add', 'view', or 'quit': ";
        cin >> action;
        cin.ignore();  // Clear newline
        if (action == "add") {
            cout << "Enter task: ";
            getline(cin, task);
            addTask(task);
        } else if (action == "view") {
            viewTasks();
        } else if (action == "quit") {
            break;
        }
    }
    return 0;
}

Project 3: Simple Calculator

Build a calculator for basic operations.

#include <iostream>
using namespace std;

void calculator() {
    double num1, num2;
    char op;
    cout << "Enter first number: ";
    cin >> num1;
    cout << "Enter operator (+, -, *, /): ";
    cin >> op;
    cout << "Enter second number: ";
    cin >> num2;
    
    switch (op) {
        case '+': cout << num1 + num2 << endl; break;
        case '-': cout << num1 - num2 << endl; break;
        case '*': cout << num1 * num2 << endl; break;
        case '/':
            if (num2 != 0) cout << num1 / num2 << endl;
            else cout << "Cannot divide by zero!" << endl;
            break;
        default: cout << "Invalid operator!" << endl;
    }
}

int main() {
    calculator();
    return 0;
}
