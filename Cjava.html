Complete Java Course for Beginners

Course Overview

This course is designed for absolute beginners with no prior programming experience. By the end, you'll be able to write Java programs, understand core programming concepts, and build simple projects. The course is hands-on, with examples and exercises to reinforce learning.

  • Duration: Approximately 4-6 weeks (assuming 2-3 hours per week)
  • Prerequisites: None
  • Tools Needed:
    • Java Development Kit (JDK) installed (download from oracle.com or use OpenJDK).
    • A code editor or IDE (e.g., IntelliJ IDEA, Eclipse, or VS Code).
    • Optional: Online platforms like Replit or JDoodle for practice.

Module 1: Introduction to Java

1.1 What is Java?

Java is a high-level, object-oriented programming language known for its portability and robustness. It is widely used for web applications, mobile apps (Android), enterprise software, and more.

1.2 Setting Up Your Environment

Install the JDK and verify by running java -version and javac -version in the terminal. Set up an IDE like IntelliJ IDEA or Eclipse for easier coding.

1.3 Writing Your First Program

Create a file named HelloWorld.java (file name must match class name) and write:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Explanation: public static void main is the entry point, and System.out.println outputs text to the console.

Compile and Run: Compile with javac HelloWorld.java (creates HelloWorld.class) and run with java HelloWorld.

1.4 Using an IDE

Use IntelliJ IDEA or Eclipse for a streamlined experience. Practice compiling and running HelloWorld.java in your IDE.

Exercise:
  1. Write a program that prints your name.
  2. Modify the program to print "Hello, [Your Name]!".

Module 2: Java Basics

2.1 Variables and Data Types

Variables must be declared with a type. Common types: int, double, char, boolean, String.

public class Variables {
    public static void main(String[] args) {
        String name = "Alice";  // String
        int age = 25;          // Integer
        double height = 5.5;   // Double
        boolean isStudent = true; // Boolean

        System.out.println(name + " " + age + " " + height + " " + isStudent);
    }
}

2.2 Basic Operations

Arithmetic: +, -, *, /, % (modulus), increment (++), decrement (--).

public class Operations {
    public static void main(String[] args) {
        int a = 10, b = 3;
        System.out.println("Sum: " + (a + b));     // 13
        System.out.println("Product: " + (a * b));  // 30
        System.out.println("Division: " + (a / b)); // 3 (integer division)
        System.out.println("Modulus: " + (a % b));  // 1
    }
}

2.3 Strings

Strings are objects in Java, supporting concatenation and methods like length().

public class Strings {
    public static void main(String[] args) {
        String greeting = "Hello";
        String name = "Bob";
        String message = greeting + ", " + name + "!";
        System.out.println(message);        // Output: Hello, Bob!
        System.out.println(greeting.charAt(0)); // Output: H
        System.out.println(name.length());   // Output: 3
    }
}

Module 3: Control Flow

3.1 Conditionals (if, else if, else)

Use if statements for decision-making.

import java.util.Scanner;

public class Conditionals {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter your age: ");
        int age = scanner.nextInt();
        if (age >= 18) {
            System.out.println("You are an adult.");
        } else if (age >= 13) {
            System.out.println("You are a teenager.");
        } else {
            System.out.println("You are a child.");
        }
        scanner.close();
    }
}

3.3 Loops

For Loop: Iterate over a range.

public class ForLoop {
    public static void main(String[] args) {
        for (int i = 0; i < 5; i++) {
            System.out.println(i);  // Prints 0 to 4
        }
    }
}

While Loop: Repeats while a condition is true.

public class WhileLoop {
    public static void main(String[] args) {
        int count = 1;
        while (count <= 5) {
            System.out.println(count);
            count++;
        }
    }
}

3.4 Break and Continue

break: Exits a loop.
continue: Skips the current iteration.

public class BreakContinue {
    public static void main(String[] args) {
        for (int i = 0; i < 10; i++) {
            if (i == 5) break;  // Stops at 5
            System.out.println(i);
        }
        for (int i = 0; i < 10; i++) {
            if (i % 2 == 0) continue;  // Skips even numbers
            System.out.println(i);
        }
    }
}

Module 4: Data Structures

4.1 Arrays

Arrays store fixed-size sequences of elements.

public class Arrays {
    public static void main(String[] args) {
        int[] numbers = {1, 2, 3, 4, 5};
        System.out.println(numbers[0]);  // Output: 1
        numbers[1] = 10;  // Modify element
        System.out.println(numbers[1]);  // Output: 10
    }
}

4.2 ArrayList

ArrayList (from java.util) is a dynamic array.

import java.util.ArrayList;

public class ArrayListExample {
    public static void main(String[] args) {
        ArrayList<String> fruits = new ArrayList<>();
        fruits.add("apple");
        fruits.add("banana");
        fruits.add("cherry");
        System.out.println(fruits.get(0));  // Output: apple
        fruits.remove(1);  // Remove banana
        for (String fruit : fruits) {
            System.out.println(fruit);  // Output: apple, cherry
        }
    }
}

4.3 HashMap

HashMap stores key-value pairs.

import java.util.HashMap;

public class HashMapExample {
    public static void main(String[] args) {
        HashMap<String, Integer> student = new HashMap<>();
        student.put("Alice", 20);
        student.put("Bob", 22);
        System.out.println(student.get("Alice"));  // Output: 20
        student.put("Charlie", 21);  // Add key-value pair
        for (String name : student.keySet()) {
            System.out.println(name + ": " + student.get(name));
        }
    }
}

public class Introduce {

    }

5.4 Scope

Variables have local or instance scope.

public class Scope {
    static int x = 10;  // Class-level (static)

    public static void myMethod() {
        int x = 5;  // Local
        System.out.println(x);  // Output: 5
    }

    public static void main(String[] args) {
        myMethod();
        System.out.println(x);  // Output: 10
    }
}

Module 6: Working with Files

6.1 Reading Files

Use BufferedReader or Files for file reading.

import java.io.BufferedReader;
import java.io.FileReader;

public class ReadFile {
    public static void main(String[] args) {
        try (BufferedReader reader = new BufferedReader(new FileReader("example.txt"))) {
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println(line);
            }
        } catch (Exception e) {
            System.out.println("Error: " + e.getMessage());
        }
    }
}

6.2 Writing Files

Use BufferedWriter or Files for writing.

import java.io.BufferedWriter;
import java.io.FileWriter;

public class WriteFile {
    public static void main(String[] args) {
        try (BufferedWriter writer = new BufferedWriter(new FileWriter("example.txt"))) {
            writer.write("Hello, Java!\n");
        } catch (Exception e) {
            System.out.println("Error: " + e.getMessage());
        }

        try (BufferedWriter writer = new BufferedWriter(new FileWriter("example.txt", true))) {
            writer.write("Appending this line.\n");
        } catch (Exception e) {
            System.out.println("Error: " + e.getMessage());
        }
    }
}

Module 7: Exception Handling

Handle errors using try-catch.

import java.util.Scanner;

public class Exceptions {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        try {
            System.out.print("Enter a number: ");
            int number = scanner.nextInt();
            System.out.println(100 / number);
        } catch (ArithmeticException e) {
            System.out.println("Cannot divide by zero!");
        } catch (Exception e) {
            System.out.println("Invalid input!");
        } finally {
            scanner.close();
        }
    }
}
